# Commandes du projet

## Développement
```bash
npm run dev       # Lance le serveur de développement (Vite) → http://localhost:5173
```

## Production
```bash
npm run build     # Compile le projet pour la production (dossier dist/)
npm run preview   # Prévisualise le build de production en local
```

## Dépendances
```bash
npm install       # Installe les dépendances
```

## Notes
- Pas de lint configuré (ESLint non installé)
- Stack : React 18 + Vite 6 + Tailwind CSS 3
- JSX uniquement, pas de TypeScript
